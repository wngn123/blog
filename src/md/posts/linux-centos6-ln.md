## 软连接语法
```
ln -s sourcepath targetpath
```
## ant添加软连接
```
ln -s /opt/dev/ant/apache-ant-1.9.4/bin/ant /usr/local/bin/ant
```
## maven添加软连接
```
ln -s /opt/dev/maven/apache-maven-3.3.3/bin/mvn /usr/local/bin/mvn
```
